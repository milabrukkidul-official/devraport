📖 Cara Setup Google Spreadsheet

  1. Buka Google Sheets dan buat spreadsheet baru.
  2. Buat sheet dengan nama: SETTING, DATA SISWA, REKAP NILAI, KKM, EKSTRAKURIKULER
  3. Buka Extensions → Apps Script, paste kode dari file gas/Code.gs
  4. Deploy sebagai Web App (Execute as: Me, Who has access: Anyone)
  5. Copy URL Web App dan paste di kolom di atas.
