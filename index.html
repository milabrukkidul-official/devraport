⚠️ URL sudah terpasang di kode

URL Apps Script sudah di-hardcode di file js/api.js baris pertama (konstanta GAS_URL). Tidak perlu diubah lagi setelah deploy.

Kolom di bawah hanya untuk override sementara (tersimpan di browser ini saja).

📖 Cara Setup (sekali saja)

  1. Buka Google Sheets → Extensions → Apps Script
  2. Paste seluruh isi file gas/Code.gs
  3. Jalankan fungsi setupSheets() sekali
  4. Deploy → New Deployment → Web App
    Execute as: Me | Access: Anyone
  5. Copy URL Web App → paste ke konstanta GAS_URL di js/api.js
  6. Commit & push ke GitHub — selesai, tidak perlu diubah lagi

📤 Upload Data Siswa (Excel)

Rombel Tujuan: -

⚠️ PENTING:

  • Pilih rombel tujuan terlebih dahulu di halaman Data Siswa
  • Download template untuk rombel tersebut (tombol 📥 Template)
  • Isi data siswa di file Excel
  • Upload file - siswa akan masuk ke rombel yang dipilih

📋 Format File:

Upload file .xlsx sesuai template. Kolom: NISN, No Induk, Nama Siswa, Nama Panggilan, Tempat Lahir, Tanggal Lahir, Nama Orang Tua

Tanggal Lahir isi sebagai teks format YYYY-MM-DD (contoh: 2010-05-17) atau format tanggal Excel biasa.

📤 Upload Rekap Nilai (Excel)

Upload file .xlsx sesuai template. Kolom pertama Nama Siswa, lalu kolom nilai tiap mata pelajaran, diakhiri Sakit, Ijin, Alpa.

Gunakan tombol 📥 Template untuk mengunduh template yang sudah berisi nama siswa dan mata pelajaran.
